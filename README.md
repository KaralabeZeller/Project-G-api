# Project-G API

## Development

### Checking-out the project
```bash
git clone https://github.com/KaralabeZeller/Project-G-api.git
```

### Running unit and integration tests
```bash
mvn verify
```

### Running the application locally
```bash
mvn spring-boot:run
```
